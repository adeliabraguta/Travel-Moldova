# 🌍 Travel Moldova
A small project developed in Vue.js that awakens the desire to start a journey thorugh Moldova. 

## ⚙️ Tech stack
✔️ VueJs <br>
✔️ Pinia <br>
✔️ JavaScript <br>
✔️ HTML, SCSS

## 💻 Instalation
1. Download the project
2. Run command:
```npm install```
```npm run dev```
